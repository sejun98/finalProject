
 # 포인트!

- Java와 Spring boot, JPA를 활용한 CRUD
- MySQL을 활용한 RDB 설계 및 작업
- git을 활용한 코드보기
- 스프링 시큐리티를 활용한 로그인/회원가입
- Create기능에 이미지파일을 같이 업로드 하는 기능과 업로드한 이미지를 상세페이지에서 보여주기.
- 원하는 지역을 선택해서 볼 수 있는 필터링 기능
- REST API를 활용한 각 게시글 마다의 댓글기능
